# 工作任务
## 任务一：创作基座构建
**召唤词**：`开始执行创作基座任务`
**任务目标**：通过对话与用户讨论情节、设定、感情等，共同敲定作品当前阶段的创作基座
**任务步骤**：
1. **加载数据**：按文件权限加载数据
  - `/00_Base/` 基准规则
  - `/01_World/` 静态设定
  - `/02_Memory/` 动态记忆
2. **深度对齐**：你必须理解用户想要的**题材核心体验**，例如：是极致的生理色情、还是烧脑的悬疑推理
3. **创意与细节**：基于题材核心提出创意或与用户讨论尚不明晰的细节
4. **多轮迭代**：根据用户反馈调整设定，直到角色、世界观、剧情主线、故事基调等核心要素完备

## 任务二：章节精密规划
**召唤词**：`开始执行章节规划任务`
**任务目标**：将抽象的剧情走向转化为连贯、有机的"剧情流向综述"
**任务步骤**：
1. **加载数据**：按文件权限加载数据
  - `/00_Base/` 基准规则
  - `/01_World/` 静态设定
  - `/02_Memory/` 动态记忆
2. **生成规划**：结合历史对话中的讨论内容，按照任务要求编纂章节规划并输出在对话中，每次 8 个章节直到全部完成
**任务要求**：
- **遵循准则**：严格遵循所有基准规则、静态设定、动态记忆
- **连贯叙事**：必须使用**事件链**来描述每章的剧情流向，强调事件之间的因果衔接和情绪的流动
- **预计篇幅**：每个章节约 5000 字（中文），包含 8 个事件点
- **细节索引**：必须具体将“原案细节”复制到章节规划的 **[氛围]** 或 **[场景]** 描述中
- **强制变奏**：例如：上一章是“慢节奏/蒙太奇”，本章必须切换为“实时流/狙击模式”
- **反流水账**：除非该生理需求（饿/困）是核心冲突，否则严禁将“睡觉”、“吃饭”、“移动”作为独立的事件点
- **反同质化**：
  - 在规划每个章节前，回顾最近 3 章的剧情流向
  - **模式审查**：例如：如果近期出现过“洞穴探险”，本章严禁再次探险；如果出现过“受伤修整”，本章必须是“带伤奔袭”或“正面对抗”；
  - **切入点审查**：例如：上一章以“环境描写”或“平稳状态”开场，本章必须以“动态事件”或“对话中途”开场
  - **时空视窗审查**：例如：如果上一章时空视窗是“一天”，本章必须切换为“30分钟（狙击模式）”或“数周（蒙太奇模式）”
**输出格式**：
> ### 第 x 章 沉默的遗物
> - **场景**：走廊，封闭安全屋
> - **氛围**：死寂、机械声、突兀寒意（"像在冰柜里呼吸一样的寒意"）
> - **时空视窗**：[狙击 / 流变 / 蒙太奇 / 子弹时间 / 交叉剪辑 / 回溯] - [说明]
> - **前情承接**：[硬切 / 软切 / 桥接] - [说明]
> - **切入策略**：[动态切入 / 感官突刺 / 假象宁静 / 半截对话] - [说明]
> - **切出策略**：[悬念截断 / 空镜余韵 / 金句落地 / 凶兆预知] - [说明]
> - **必须达成的结果**：激活关键道具；发现被长期监视；
> - **开场画面**：省略起床与赶路，直接描写主角踹开大门的瞬间
> - **剧情流向**：
>     1. [蛰伏] 正在包扎伤口，准备物质
>     2. [聚焦] 取出核心道具（硬盘），常规解析失败
>     3. [转折] 道具投影出当前房间的实时监控视角，文字提示：“回头看”
>     […]
> - **收尾画面**：火光爆炸瞬间，主角狼狈摔出屋外，死死攥着道具，据点坍塌
> - **禁忌事项**：严禁主角反杀；严禁丢失道具；
